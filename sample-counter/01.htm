CSS property: counter

The example below is based on David Storey's Counters to replace the start attribute. There are adjustments I'd like to make:

  • Using a class instead of nth-of-type gives you much more control over when numbering starts and finishes
  • Adding a float to the generated list number means that all list items can be forced to line up neatly at the left edge - regardless of how long the generated number is.
