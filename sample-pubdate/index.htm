HTML attribute: pubdate

Note: W3C and WHATWG groups have dropped pubdate attribute from specification of time element. It should be removed from this module too. See official pages:

  • http://www.whatwg.org/specs/web-apps/current-work/multipage/text-level-semantics.html#the-time-element
  • http://www.w3.org/TR/html-markup/time.html

The pubdate attribute allows authors to define that the date being used is the date of publish.
