Selector: class

Example 1 - case sensitive

This example shows that class names are case sensitive: 

.intro { color: red; }
.Intro { color: blue; }

This is a paragraph with a class of "intro"

This is a paragraph with a class of "Intro"

Example 2 - refining class selectors using elements

This example shows that elements can be added before class names to make them more specific: 

p.intro { color: red; }

This is a paragraph with a class of "intro"

This is a div with a class of "intro"

Example 3 - multiple class strings

Classes and be added together to form subsets. 

.test1 { color: red; }
.test2 { color: blue; }
.test1.test2 { color: green; }

This is a paragraph with a class of "test1"

This is a paragraph with a class of "test2"

This is a paragraph with a class of "test1 test2"

Example 4 - multiple class strings

The order of class names is irrelevant: 

.class2.class1 { color: red; }

This is a paragraph with a class of "class1 class2"

Example 5 - multiple class strings

The order of HTML attributes is irrelevant: 

.class1.class2 { color: red; }

This is a paragraph with a class of "class1 class2"

This is a paragraph with a class of "class2 class1"
