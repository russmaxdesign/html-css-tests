Example 2 - longdesc attribute

In this example, the long description is defined using the "longdesc" attribute that points to a new page with the long description. This attribute is supported by some assistive devices, but the long description is not available to everyone.

Example 3 - longdesc attribute with visible "D" link

In this example, the long description is defined again using the "longdesc" attribute that points to a new page with the long description. However, this example also includes a visible "D" link, which allows users to click and then read the long description.

Example 4 - aria-describedby

In this example, the long description is placed away from the image - there is non-related content between the image and long description. However, the image and it's long description are linked via the "aria-describedby" attribute and it's associated value "longdesc4".
