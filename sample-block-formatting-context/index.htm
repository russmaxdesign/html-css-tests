CSS: The block formatting context

A block formatting context is a box that has been styled with at least one of the following:

  • float: left
  • float: right
  • overflow: hidden
  • overflow: auto
  • overflow: scroll
  • display: table-cell
  • table-caption
  • inline-block
  • position: absolute
  • position: fixed

The positioning scheme to which block formatting contexts belong is normal flow. They behave more or less like any block box, apart from these important exceptions:

  • Block formatting contexts prevent margin collapsing
  • Block formatting contexts contain floats
  • Block formatting contexts do not overlap floats
