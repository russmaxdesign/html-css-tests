CSS: percentage width and height

Example 1

When the parent element has been given a height, a child element given a percentage height will take this height front the parent.

Child with pink background - inside the parent with yellow background

Example 2

When the parent element has been given no height, a child element given a percentage height will attempt take this height from the parent. However, as the parent has no height, the child's height will not be applied

Child with pink background - inside the parent with yellow background

Example 3

When the parent element has been given a width, a child element given a percentage width will take this width front the parent.

Child with pink background - inside the parent with yellow background

Example 4

When the parent element has been given a width, and a child element has been given a percentage width and percentage padding-left, both the width and padding-left will be in relation to the parent's width.

Child with lime background - inside the parent with yellow background

Example 5

A child element given a percentage-based padding-bottom will take this measurement from it's width.

Child with lime background - inside the parent with yellow background

Example 6

A child element given a percentage-based padding-bottom will take this measurement from it's width - even if the parent has been given a height.

Child with lime background - inside the parent with yellow background
